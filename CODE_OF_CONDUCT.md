# Code de conduite

Ce projet adopte le **[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**
comme code de conduite (versions [française](https://www.contributor-covenant.org/fr/version/2/1/code_of_conduct/)
et [anglaise](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) disponibles).

En participant à ce projet (issues, pull requests, discussions), vous acceptez de
respecter ses principes : un environnement ouvert, accueillant et respectueux pour tous.

## Application

Les comportements abusifs ou inacceptables peuvent être signalés en privé au
mainteneur [@foXaCe](https://github.com/foXaCe). Toutes les plaintes seront examinées
et traitées de manière équitable et confidentielle.
